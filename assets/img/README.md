# Imagens do site

Coloque os arquivos abaixo nesta pasta. Enquanto não existirem, a página exibe placeholders com o caminho esperado.

| Arquivo | Uso |
|---------|-----|
| `avatar.jpg` | Foto no hero (pode usar `.png` ou `.webp` — ajuste o `src` em `index.html`) |
| `coleta-frai.png` | Screenshot ou mockup do app Coleta Frai |
| `slice-heads.png` | Screenshot ou mockup do app Slice Heads |

Sugestão: imagens em proporção vertical para os apps (ex.: 9:16) e quadrada para o avatar.
